Cliente-Servidor

Uma estrutura cliente-servidor é um modelo de arquitetura
de rede em que um programa de computador (cliente),
solicita um serviço ou recurso de outro de computador
(servidor), recebendo uma resposta as
informações solicitadas

O cliente é um dispositivo (desktops,
smartphones, etc.) que solicita
serviços a um servidor, o qual fornece
esses serviços. O servidor pode ser
um computador dedicado ou um
software em rede.

A rede conecta clientes e
servidores, podendo ser local
(LAN) ou global (internet).
Os protocolos de
comunicação, como HTTP,
SMTP e FTP, regem
essa interação.

O banco de dados armazena
informações usadas pelo
servidor, enquanto a API permite
que aplicativos se comuniquem
com o servidor e acessem seus
serviços.
